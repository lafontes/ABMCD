Sobre o Projeto

Esta página foi desenvolvida para apoiar uma ONG, com o objetivo de divulgar seu trabalho e facilitar o acesso às informações para o público em geral. A ideia é apresentar de forma clara quem é a organização, quais são suas ações e como as pessoas podem contribuir ou entrar em contato.

O site foi pensado para ser simples, funcional e fácil de navegar, permitindo que qualquer visitante encontre rapidamente o que procura e conheça melhor a causa defendida pela ONG.

Objetivo
Apresentar a ONG e sua missão
Divulgar projetos e ações sociais
Facilitar o contato com o público
Incentivar o engajamento e o apoio à causa

Tecnologias Utilizadas
HTML
CSS
JavaScript

Considerações Finais
Este projeto busca usar a tecnologia como uma ferramenta de apoio ao trabalho social da ONG, ajudando a ampliar sua visibilidade e alcance.
